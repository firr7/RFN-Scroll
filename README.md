# KnowScroll 📱

App per studiare la Rivoluzione Francese e l'Età Napoleonica in formato reel.

## File inclusi
- `index.html` — l'app completa
- `manifest.json` — configurazione PWA
- `sw.js` — service worker (funzionamento offline)
- `icon-192.png` / `icon-512.png` — icone app

## Come attivare GitHub Pages

1. Vai nella tua repository su GitHub
2. Clicca **Settings** (in alto a destra)
3. Nel menu a sinistra clicca **Pages**
4. Sotto "Branch" seleziona **main** e clicca **Save**
5. Dopo 1-2 minuti il sito sarà live su `https://TUONOME.github.io/NOMEREPO`

## Come installare l'app sul telefono

### iPhone (Safari)
1. Apri il link in **Safari**
2. Tocca l'icona **Condividi** (quadrato con freccia in su)
3. Scorri e tocca **"Aggiungi alla schermata Home"**
4. Tocca **Aggiungi** — l'icona appare sulla home!

### Android (Chrome)
1. Apri il link in **Chrome**
2. Tocca i tre puntini in alto a destra
3. Tocca **"Installa app"** o **"Aggiungi alla schermata Home"**
4. Conferma — l'icona appare sulla home!
